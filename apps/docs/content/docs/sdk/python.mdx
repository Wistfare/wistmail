## Configuration

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
import { WistMail } from 'wistmail'

const wm = new WistMail({
  apiKey: 'wm_your_api_key',
  baseUrl: 'https://your-wistmail.com',
})
```
</Tab>
<Tab value="Python">
```python
from wistmail import WistMail

wm = WistMail(
    api_key="wm_your_api_key",              # Required
    base_url="https://your-wistmail.com",    # Default: https://api.wistmail.com
    timeout=30.0,                            # Default: 30.0 seconds
)
```

| Option | Type | Required | Default | Description |
|--------|------|----------|---------|-------------|
| `api_key` | str | Yes | -- | Your WistMail API key |
| `base_url` | str | No | `https://api.wistmail.com` | WistMail server URL |
| `timeout` | float | No | `30.0` | Request timeout in seconds |

Supports context manager for automatic cleanup:

```python
with WistMail(api_key="wm_your_api_key") as wm:
    wm.emails.send(...)
```
</Tab>
<Tab value="cURL">
```bash
# No SDK needed. Set your base URL and API key:
export BASE_URL="https://your-wistmail.com"
export API_KEY="wm_your_api_key"
```
</Tab>
</Tabs>

## Emails

### `wm.emails.send(**kwargs)`

Send a single email. Uses keyword-only arguments.

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const { id } = await wm.emails.send({
  from: 'hello@yourdomain.com',
  to: ['user@example.com'],
  subject: 'Welcome!',
  html: '<h1>Hello!</h1>',
})
```
</Tab>
<Tab value="Python">
```python
result = wm.emails.send(
    from_address="hello@yourdomain.com",
    to=["user@example.com"],
    subject="Welcome!",
    html="<h1>Hello!</h1>",
    tags={"campaign": "onboarding"},
)

print(result["id"])  # eml_abc123
```
</Tab>
<Tab value="cURL">
```bash
curl -X POST https://your-wistmail.com/api/v1/emails \
  -H "Authorization: Bearer wm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "hello@yourdomain.com",
    "to": ["user@example.com"],
    "subject": "Welcome!",
    "html": "<h1>Hello!</h1>"
  }'
```
</Tab>
</Tabs>

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from_address` | str | Yes | Sender email (maps to `from` in the API) |
| `to` | str \| list[str] | Yes | Recipient(s) |
| `subject` | str | Yes | Subject line |
| `html` | str | No | HTML body |
| `text` | str | No | Plain text body |
| `cc` | str \| list[str] | No | CC recipients |
| `bcc` | str \| list[str] | No | BCC recipients |
| `reply_to` | str \| list[str] | No | Reply-To address(es) |
| `headers` | dict | No | Custom email headers |
| `tags` | dict | No | Key-value tags for tracking |
| `scheduled_at` | str \| datetime | No | Delayed send time |
| `template_id` | str | No | Template ID |
| `variables` | dict | No | Template variable values |

## Error Handling

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
import { ValidationError, RateLimitError, NotFoundError } from 'wistmail'

try {
  await wm.emails.send({ /* ... */ })
} catch (err) {
  if (err instanceof ValidationError) console.error(err.message, err.details)
  if (err instanceof RateLimitError) console.error(`Retry after ${err.retryAfter}s`)
}
```
</Tab>
<Tab value="Python">
```python
from wistmail.errors import (
    WistMailError,
    AuthenticationError,
    ValidationError,
    NotFoundError,
    RateLimitError,
)

try:
    wm.emails.send(...)
except ValidationError as e:
    print(f"Invalid request: {e.message}")
    print(f"Details: {e.details}")
except RateLimitError as e:
    print(f"Rate limited. Retry after {e.retry_after}s")
except NotFoundError as e:
    print(f"Not found: {e.message}")
except AuthenticationError as e:
    print(f"Auth error: {e.message}")
except WistMailError as e:
    print(f"Error {e.status_code}: {e.message}")
```
</Tab>
<Tab value="cURL">
```json
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid email address format"
  }
}
```
</Tab>
</Tabs>

| Exception | HTTP Status | Description |
|-----------|-------------|-------------|
| `AuthenticationError` | 401 | Invalid or missing API key |
| `ValidationError` | 400 | Invalid request parameters |
| `NotFoundError` | 404 | Resource does not exist |
| `RateLimitError` | 429 | Too many requests |
| `WistMailError` | Other | Base exception for all errors |
