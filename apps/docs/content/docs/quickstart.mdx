## 1. Get Your API Key

After setting up WistMail, create an API key from **Settings > API Keys** in the web dashboard, or use the API:

```bash title="Create an API key"
curl -X POST https://your-wistmail.com/api/v1/api-keys \
  -H "Cookie: wm_session=your_session_token" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My App",
    "scopes": ["emails:send", "emails:read"]
  }'
```

Your API key will look like: `wm_a1b2c3d4e5f6...`

## 3. Send Your First Email

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
import { WistMail } from 'wistmail'

const wm = new WistMail({
  apiKey: 'wm_your_api_key',
  baseUrl: 'https://your-wistmail.com',
})

const { id } = await wm.emails.send({
  from: 'hello@yourdomain.com',
  to: 'recipient@example.com',
  subject: 'Hello from WistMail!',
  html: '<h1>It works!</h1><p>Your email infrastructure is live.</p>',
})

console.log('Email sent:', id) // eml_abc123...
```
</Tab>
<Tab value="Python">
```python
from wistmail import WistMail

wm = WistMail(
    api_key="wm_your_api_key",
    base_url="https://your-wistmail.com",
)

result = wm.emails.send(
    from_address="hello@yourdomain.com",
    to="recipient@example.com",
    subject="Hello from WistMail!",
    html="<h1>It works!</h1><p>Your email infrastructure is live.</p>",
)

print("Email sent:", result["id"])  # eml_abc123...
```
</Tab>
<Tab value="cURL">
```bash
curl -X POST https://your-wistmail.com/api/v1/emails \
  -H "Authorization: Bearer wm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "hello@yourdomain.com",
    "to": "recipient@example.com",
    "subject": "Hello from WistMail!",
    "html": "<h1>It works!</h1><p>Your email infrastructure is live.</p>"
  }'
```
</Tab>
</Tabs>

## 4. Check Email Status

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const status = await wm.emails.get('eml_abc123')
console.log(status.status) // 'delivered'
console.log(status.deliveredAt) // '2026-03-28T12:00:00Z'
```
</Tab>
<Tab value="Python">
```python
status = wm.emails.get("eml_abc123")
print(status["status"])  # 'delivered'
print(status["deliveredAt"])  # '2026-03-28T12:00:00Z'
```
</Tab>
<Tab value="cURL">
```bash
curl https://your-wistmail.com/api/v1/emails/eml_abc123 \
  -H "Authorization: Bearer wm_your_api_key"
```
</Tab>
</Tabs>
