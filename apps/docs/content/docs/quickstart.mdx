## 1. Get Your API Key

Log in to the Wistfare Mail dashboard, go to **Settings > API Keys**, and create a new key with the `emails:send` scope. Your key will look like `wm_a1b2c3d4e5f6...` — copy it, you will only see it once.

## 2. Install the SDK

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```bash
npm install wistmail
```
</Tab>
<Tab value="Python">
```bash
pip install wistmail
```
</Tab>
<Tab value="cURL">
No installation needed — use any HTTP client.
</Tab>
</Tabs>

## 3. Send Your First Email

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
import { WistMail } from 'wistmail'

const wm = new WistMail({
  apiKey: 'wm_your_api_key',
  baseUrl: 'https://mail.wistfare.com',
})

const { id } = await wm.emails.send({
  from: 'hello@wistfare.com',
  to: 'recipient@example.com',
  subject: 'Hello from Wistfare Mail!',
  html: '<h1>It works!</h1><p>Your email infrastructure is live.</p>',
})

console.log('Email sent:', id) // eml_abc123...
```
</Tab>
<Tab value="Python">
```python
from wistmail import WistMail

wm = WistMail(
    api_key="wm_your_api_key",
    base_url="https://mail.wistfare.com",
)

result = wm.emails.send(
    from_address="hello@wistfare.com",
    to="recipient@example.com",
    subject="Hello from Wistfare Mail!",
    html="<h1>It works!</h1><p>Your email infrastructure is live.</p>",
)

print("Email sent:", result["id"])  # eml_abc123...
```
</Tab>
<Tab value="cURL">
```bash
curl -X POST https://mail.wistfare.com/api/v1/emails \
  -H "X-API-Key: wm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "hello@wistfare.com",
    "to": "recipient@example.com",
    "subject": "Hello from Wistfare Mail!",
    "html": "<h1>It works!</h1><p>Your email infrastructure is live.</p>"
  }'
```
</Tab>
</Tabs>

## 4. Check Delivery Status

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const status = await wm.emails.get('eml_abc123')
console.log(status.status)      // 'delivered'
console.log(status.deliveredAt) // '2026-03-28T12:00:00Z'
```
</Tab>
<Tab value="Python">
```python
status = wm.emails.get("eml_abc123")
print(status["status"])       # 'delivered'
print(status["deliveredAt"])  # '2026-03-28T12:00:00Z'
```
</Tab>
<Tab value="cURL">
```bash
curl https://mail.wistfare.com/api/v1/emails/eml_abc123 \
  -H "X-API-Key: wm_your_api_key"
```
</Tab>
</Tabs>

## 5. Set Up Webhooks (Optional)

Receive real-time delivery events by creating a webhook endpoint:

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const webhook = await wm.webhooks.create({
  url: 'https://yourapp.com/webhooks/email',
  events: ['email.delivered', 'email.bounced', 'email.opened'],
})

console.log(webhook.secret) // whsec_... — use this to verify signatures
```
</Tab>
<Tab value="Python">
```python
webhook = wm.webhooks.create(
    url="https://yourapp.com/webhooks/email",
    events=["email.delivered", "email.bounced", "email.opened"],
)

print(webhook["secret"])  # whsec_... — use this to verify signatures
```
</Tab>
<Tab value="cURL">
```bash
curl -X POST https://mail.wistfare.com/api/v1/webhooks \
  -H "X-API-Key: wm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://yourapp.com/webhooks/email",
    "events": ["email.delivered", "email.bounced", "email.opened"]
  }'
```
</Tab>
</Tabs>
