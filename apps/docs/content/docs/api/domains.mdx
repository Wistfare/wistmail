## Add a Domain

`POST /api/v1/domains`

**Scope**: `domains:manage`

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Domain name (e.g., `yourdomain.com`) |

### Example

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const domain = await wm.domains.create('yourdomain.com')
console.log(domain.id) // dom_abc123
console.log(domain.dnsRecords) // DNS records to add
```
</Tab>
<Tab value="Python">
```python
domain = wm.domains.create("yourdomain.com")
print(domain["id"])  # dom_abc123
print(domain["dnsRecords"])  # DNS records to add
```
</Tab>
<Tab value="cURL">
```bash
curl -X POST https://your-wistmail.com/api/v1/domains \
  -H "Authorization: Bearer wm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"name": "yourdomain.com"}'
```
</Tab>
</Tabs>

### Response

```json
{
  "id": "dom_abc123",
  "name": "yourdomain.com",
  "status": "pending",
  "dnsRecords": [
    {
      "type": "MX",
      "name": "yourdomain.com",
      "value": "mail.your-wistmail.com",
      "priority": 10
    },
    {
      "type": "TXT",
      "name": "yourdomain.com",
      "value": "v=spf1 include:your-wistmail.com ~all"
    },
    {
      "type": "TXT",
      "name": "wm._domainkey.yourdomain.com",
      "value": "v=DKIM1; k=rsa; p=MIGfMA0GCSqGSIb3DQEBAQUA..."
    },
    {
      "type": "TXT",
      "name": "_dmarc.yourdomain.com",
      "value": "v=DMARC1; p=quarantine; rua=mailto:dmarc@your-wistmail.com"
    }
  ],
  "createdAt": "2026-03-28T00:00:00Z"
}
```

### DNS Records

After creating a domain, add these records to your DNS provider:

| Record | Purpose | Description |
|--------|---------|-------------|
| **MX** | Mail routing | Routes incoming mail to your WistMail server |
| **SPF** (TXT) | Sender verification | Authorizes WistMail to send on your behalf |
| **DKIM** (TXT) | Email signing | Cryptographic signature for email authenticity |
| **DMARC** (TXT) | Policy | Tells receivers how to handle failed checks |

## Verify a Domain

`POST /api/v1/domains/:id/verify`

**Scope**: `domains:manage`

Triggers a DNS lookup to check if all required records are configured correctly.

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const result = await wm.domains.verify('dom_abc123')
console.log(result.status) // 'verified' or 'pending'
console.log(result.checks) // { mx: true, spf: true, dkim: false, dmarc: true }
```
</Tab>
<Tab value="Python">
```python
result = wm.domains.verify("dom_abc123")
print(result["status"])  # 'verified' or 'pending'
print(result["checks"])  # {'mx': True, 'spf': True, 'dkim': False, 'dmarc': True}
```
</Tab>
<Tab value="cURL">
```bash
curl -X POST https://your-wistmail.com/api/v1/domains/dom_abc123/verify \
  -H "Authorization: Bearer wm_your_api_key"
```
</Tab>
</Tabs>

### Response

```json
{
  "status": "pending",
  "checks": {
    "mx": true,
    "spf": true,
    "dkim": false,
    "dmarc": true
  },
  "message": "DKIM record not found. Please add the TXT record for wm._domainkey.yourdomain.com."
}
```

A domain is marked `verified` only when all four checks pass. DNS propagation can take up to 48 hours.

## Delete a Domain

`DELETE /api/v1/domains/:id`

**Scope**: `domains:manage`

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
await wm.domains.delete('dom_abc123')
```
</Tab>
<Tab value="Python">
```python
wm.domains.delete("dom_abc123")
```
</Tab>
<Tab value="cURL">
```bash
curl -X DELETE https://your-wistmail.com/api/v1/domains/dom_abc123 \
  -H "Authorization: Bearer wm_your_api_key"
```
</Tab>
</Tabs>

Returns `204 No Content` on success.

### Domain Statuses

| Status | Description |
|--------|-------------|
| `pending` | Domain created, DNS records not yet verified |
| `verified` | All DNS records confirmed, domain ready for sending |
| `failed` | Verification attempted but one or more records missing |
