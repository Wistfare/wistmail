## Events

WistMail can send webhook notifications for the following events:

| Event | Description |
|-------|-------------|
| `email.sent` | Email accepted and sent to recipient server |
| `email.delivered` | Email confirmed delivered to inbox |
| `email.bounced` | Email delivery failed (hard or soft bounce) |
| `email.opened` | Recipient opened the email |
| `email.clicked` | Recipient clicked a link in the email |
| `email.complained` | Recipient marked the email as spam |
| `email.failed` | Email sending failed due to an error |
| `email.received` | Incoming email received on a verified domain |

## Create a Webhook

`POST /api/v1/webhooks`

**Scope**: `webhooks:manage`

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `url` | string | Yes | HTTPS endpoint to receive events |
| `events` | string[] | Yes | List of events to subscribe to |

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const webhook = await wm.webhooks.create({
  url: 'https://yourapp.com/webhooks/wistmail',
  events: ['email.delivered', 'email.bounced', 'email.opened'],
})

console.log(webhook.id) // whk_abc123
console.log(webhook.signingSecret) // whsec_... (shown once)
```
</Tab>
<Tab value="Python">
```python
webhook = wm.webhooks.create(
    url="https://yourapp.com/webhooks/wistmail",
    events=["email.delivered", "email.bounced", "email.opened"],
)

print(webhook["id"])  # whk_abc123
print(webhook["signingSecret"])  # whsec_... (shown once)
```
</Tab>
<Tab value="cURL">
```bash
curl -X POST https://your-wistmail.com/api/v1/webhooks \
  -H "Authorization: Bearer wm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://yourapp.com/webhooks/wistmail",
    "events": ["email.delivered", "email.bounced", "email.opened"]
  }'
```
</Tab>
</Tabs>

### Response

```json
{
  "id": "whk_abc123",
  "url": "https://yourapp.com/webhooks/wistmail",
  "events": ["email.delivered", "email.bounced", "email.opened"],
  "signingSecret": "whsec_a1b2c3d4e5f6...",
  "active": true,
  "createdAt": "2026-03-28T00:00:00Z"
}
```

The `signingSecret` is only returned at creation time. Store it securely for signature verification.

## Webhook Payload

When an event fires, WistMail sends a `POST` request to your endpoint with this payload:

```json
{
  "id": "evt_abc123",
  "type": "email.delivered",
  "createdAt": "2026-03-28T12:00:00Z",
  "data": {
    "emailId": "eml_abc123",
    "from": "hello@yourdomain.com",
    "to": "user@example.com",
    "subject": "Welcome!",
    "deliveredAt": "2026-03-28T12:00:01Z"
  }
}
```

Your endpoint must return a `2xx` status code within 10 seconds. Failed deliveries are retried with exponential backoff up to 3 times.

## Signature Verification

Every webhook request includes a `Wistmail-Signature` header for verifying authenticity. The signature is an HMAC-SHA256 hex digest of the raw request body using your `signingSecret`.

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
import crypto from 'node:crypto'

function verifyWebhook(body: string, signature: string, secret: string): boolean {
  const expected = crypto
    .createHmac('sha256', secret)
    .update(body)
    .digest('hex')
  return crypto.timingSafeEqual(
    Buffer.from(signature),
    Buffer.from(expected),
  )
}

// In your request handler:
const body = await request.text()
const signature = request.headers.get('Wistmail-Signature')
const isValid = verifyWebhook(body, signature, 'whsec_your_signing_secret')

if (!isValid) {
  return new Response('Invalid signature', { status: 401 })
}

const event = JSON.parse(body)
// Handle event...
```
</Tab>
<Tab value="Python">
```python
import hmac
import hashlib

def verify_webhook(body: bytes, signature: str, secret: str) -> bool:
    expected = hmac.new(
        secret.encode(),
        body,
        hashlib.sha256,
    ).hexdigest()
    return hmac.compare_digest(signature, expected)

# In your request handler (Flask example):
body = request.get_data()
signature = request.headers.get("Wistmail-Signature")
is_valid = verify_webhook(body, signature, "whsec_your_signing_secret")

if not is_valid:
    return "Invalid signature", 401

event = request.get_json()
# Handle event...
```
</Tab>
<Tab value="cURL">
```bash
# Compute the expected signature for testing:
echo -n '{"id":"evt_abc123","type":"email.delivered"}' \
  | openssl dgst -sha256 -hmac "whsec_your_signing_secret"

# Compare with the Wistmail-Signature header value
```
