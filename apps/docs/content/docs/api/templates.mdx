## Create a Template

`POST /api/v1/templates`

**Scope**: `templates:manage`

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Template name for internal reference |
| `subject` | string | Yes | Email subject line (supports variables) |
| `html` | string | Yes | HTML body (supports variables) |
| `variables` | array | No | Variable definitions with defaults |

Variables use double curly braces: `{{variableName}}`.

### Example

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const template = await wm.templates.create({
  name: 'Welcome Email',
  subject: 'Welcome, {{name}}!',
  html: `
    <h1>Welcome, {{name}}!</h1>
    <p>Thanks for joining {{company}}.</p>
    <a href="{{dashboardUrl}}">Go to Dashboard</a>
  `,
  variables: [
    { name: 'name', required: true },
    { name: 'company', defaultValue: 'WistMail' },
    { name: 'dashboardUrl', required: true },
  ],
})

console.log(template.id) // tpl_abc123
```
</Tab>
<Tab value="Python">
```python
template = wm.templates.create(
    name="Welcome Email",
    subject="Welcome, {{name}}!",
    html="""
    <h1>Welcome, {{name}}!</h1>
    <p>Thanks for joining {{company}}.</p>
    <a href="{{dashboardUrl}}">Go to Dashboard</a>
    """,
    variables=[
        {"name": "name", "required": True},
        {"name": "company", "defaultValue": "WistMail"},
        {"name": "dashboardUrl", "required": True},
    ],
)

print(template["id"])  # tpl_abc123
```
</Tab>
<Tab value="cURL">
```bash
curl -X POST https://your-wistmail.com/api/v1/templates \
  -H "Authorization: Bearer wm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Welcome Email",
    "subject": "Welcome, {{name}}!",
    "html": "<h1>Welcome, {{name}}!</h1><p>Thanks for joining {{company}}.</p>",
    "variables": [
      { "name": "name", "required": true },
      { "name": "company", "defaultValue": "WistMail" },
      { "name": "dashboardUrl", "required": true }
    ]
  }'
```
</Tab>
</Tabs>

### Response

```json
{
  "id": "tpl_abc123",
  "name": "Welcome Email",
  "subject": "Welcome, {{name}}!",
  "html": "<h1>Welcome, {{name}}!</h1>...",
  "variables": [
    { "name": "name", "required": true },
    { "name": "company", "defaultValue": "WistMail" },
    { "name": "dashboardUrl", "required": true }
  ],
  "createdAt": "2026-03-28T00:00:00Z",
  "updatedAt": "2026-03-28T00:00:00Z"
}
```

### Sending with a Template

Use the `templateId` and `variables` fields when sending an email:

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
await wm.emails.send({
  from: 'hello@yourdomain.com',
  to: 'user@example.com',
  templateId: 'tpl_abc123',
  variables: {
    name: 'Alice',
    dashboardUrl: 'https://app.example.com/dashboard',
  },
})
```
</Tab>
<Tab value="Python">
```python
wm.emails.send(
    from_address="hello@yourdomain.com",
    to="user@example.com",
    template_id="tpl_abc123",
    variables={
        "name": "Alice",
        "dashboardUrl": "https://app.example.com/dashboard",
    },
)
```
</Tab>
<Tab value="cURL">
```bash
curl -X POST https://your-wistmail.com/api/v1/emails \
  -H "Authorization: Bearer wm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "hello@yourdomain.com",
    "to": "user@example.com",
    "templateId": "tpl_abc123",
    "variables": {
      "name": "Alice",
      "dashboardUrl": "https://app.example.com/dashboard"
    }
  }'
```
</Tab>
</Tabs>

When using a template, the `subject` and `html` fields are optional -- they come from the template. Any provided `variables` override template defaults.

## List Templates

`GET /api/v1/templates`

**Scope**: `templates:manage`

Supports pagination with `page` and `pageSize` query parameters.

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const { data, total, page, pageSize } = await wm.templates.list({
  page: 1,
  pageSize: 10,
})

data.forEach(t => console.log(t.name))
```
</Tab>
<Tab value="Python">
```python
result = wm.templates.list(page=1, page_size=10)
for t in result["data"]:
    print(t["name"])
```
</Tab>
<Tab value="cURL">
```bash
curl "https://your-wistmail.com/api/v1/templates?page=1&pageSize=10" \
  -H "Authorization: Bearer wm_your_api_key"
```
</Tab>
</Tabs>

### Response

```json
{
  "data": [
    {
      "id": "tpl_abc123",
      "name": "Welcome Email",
      "subject": "Welcome, {{name}}!",
      "createdAt": "2026-03-28T00:00:00Z",
      "updatedAt": "2026-03-28T00:00:00Z"
    }
  ],
  "total": 1,
  "page": 1,
  "pageSize": 10
}
```

## Update a Template

`PATCH /api/v1/templates/:id`

**Scope**: `templates:manage`

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | No | Updated template name |
| `subject` | string | No | Updated subject line |
| `html` | string | No | Updated HTML body |
| `variables` | array | No | Updated variable definitions |

All fields are optional -- only provided fields are updated.

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const updated = await wm.templates.update('tpl_abc123', {
  subject: 'Welcome aboard, {{name}}!',
  html: '<h1>Welcome aboard, {{name}}!</h1>',
})
```
</Tab>
<Tab value="Python">
```python
updated = wm.templates.update(
    "tpl_abc123",
    subject="Welcome aboard, {{name}}!",
    html="<h1>Welcome aboard, {{name}}!</h1>",
)
```
</Tab>
<Tab value="cURL">
```bash
curl -X PATCH https://your-wistmail.com/api/v1/templates/tpl_abc123 \
  -H "Authorization: Bearer wm_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "subject": "Welcome aboard, {{name}}!",
    "html": "<h1>Welcome aboard, {{name}}!</h1>"
  }'
```
</Tab>
</Tabs>

## Delete a Template

`DELETE /api/v1/templates/:id`

**Scope**: `templates:manage`

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
await wm.templates.delete('tpl_abc123')
```
</Tab>
<Tab value="Python">
```python
wm.templates.delete("tpl_abc123")
```
</Tab>
<Tab value="cURL">
```bash
curl -X DELETE https://your-wistmail.com/api/v1/templates/tpl_abc123 \
  -H "Authorization: Bearer wm_your_api_key"
```
</Tab>
</Tabs>

Returns `204 No Content` on success.
