## Overview

`GET /api/v1/analytics/overview`

**Scope**: `analytics:read`

Returns aggregate delivery and engagement metrics for your account.

<Tabs items={["Node.js", "Python", "cURL"]}>
<Tab value="Node.js">
```typescript
const stats = await wm.analytics.overview()

console.log(stats.sent)         // 15230
console.log(stats.delivered)    // 14950
console.log(stats.deliveryRate) // 0.9816
console.log(stats.openRate)     // 0.4523
console.log(stats.clickRate)    // 0.1207
console.log(stats.bounceRate)   // 0.0184
```
</Tab>
<Tab value="Python">
```python
stats = wm.analytics.overview()
```

### Response

```json
{
  "sent": 15230,
  "delivered": 14950,
  "opened": 6762,
  "clicked": 1804,
  "bounced": 280,
  "complained": 12,
  "failed": 38,
  "deliveryRate": 0.9816,
  "openRate": 0.4523,
  "clickRate": 0.1207,
  "bounceRate": 0.0184,
  "complaintRate": 0.0008,
  "period": {
    "start": "2026-03-01T00:00:00Z",
    "end": "2026-03-28T23:59:59Z"
  }
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `sent` | number | Total emails sent |
| `delivered` | number | Emails confirmed delivered |
| `opened` | number | Unique email opens |
| `clicked` | number | Unique link clicks |
| `bounced` | number | Hard and soft bounces |
| `complained` | number | Spam complaints |
| `failed` | number | Sending failures |
| `deliveryRate` | number | `delivered / sent` (0 to 1) |
| `openRate` | number | `opened / delivered` (0 to 1) |
| `clickRate` | number | `clicked / delivered` (0 to 1) |
| `bounceRate` | number | `bounced / sent` (0 to 1) |
| `complaintRate` | number | `complained / delivered` (0 to 1) |
| `period.start` | string | ISO 8601 start of reporting period |
| `period.end` | string | ISO 8601 end of reporting period |

### Metric Definitions

- **Delivery rate**: Percentage of sent emails that reached the recipient's mail server.
- **Open rate**: Percentage of delivered emails opened by recipients. Tracked via a 1x1 tracking pixel. May undercount due to email clients that block images.
- **Click rate**: Percentage of delivered emails where a recipient clicked at least one link.
- **Bounce rate**: Percentage of sent emails that bounced. Includes both hard bounces (invalid address) and soft bounces (full mailbox, temporary failures).
- **Complaint rate**: Percentage of delivered emails reported as spam. Keep this below 0.1% to maintain good deliverability.
